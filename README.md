# Pizza Delivery App

This is a MERN stack pizza delivery application that I built. It allows users to order pizzas and has an admin panel to manage orders.

## Tech Stack
* MongoDB
* Express
* React
* Node.js

## Features
* Order pizzas
* Add items to cart
* Razorpay payment integration
* Admin dashboard to check live orders
* Email confirmation

## How to run locally
1. Clone the repo
2. Run `npm install` in frontend and backend folders
3. Add your MongoDB URL and Razorpay keys to `backend/.env`
4. Run `npm run seed` to load initial pizzas
5. Run `npm run dev` in both folders
